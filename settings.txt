что бы срабатывало автообновление страницы при изменении кода надо 
- установить зависимости vite (npm install)
- настроить файл vite.config
- должен быть запущен опенсервер и vite в режиме dev на одном порту

полезные расширения для vsc для laravel и php
- PHP Intelephense, Laravel Blade Snippets

что бы полностью заработало форматирование blade файлов надо
- установить расширение prettier, указать конфигурацию (возможно prettier не нужен)
- открыть файл нажать Shift+Alt+F и установить расширение laravel blade formatter
